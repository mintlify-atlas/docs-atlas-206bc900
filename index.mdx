La Mubi is a browser-based party game built around movies. Gather a group of friends in the same room, pick a mode, and let the TMDB-powered movie engine do the rest — no accounts, no installs, no waiting.

## How it works

<Steps>
  <Step title="Get a TMDB API key">
    Create a free account at [themoviedb.org](https://www.themoviedb.org/settings/api) and generate an API key. La Mubi uses it to fetch high-rated movies for every round.
  </Step>
  <Step title="Configure and run">
    Add your key to a `.env` file, install dependencies, and start the Next.js dev server. The game is ready at `http://localhost:3000`.
  </Step>
  <Step title="Set up the lobby">
    Choose Charades or Impostor mode, configure teams and players, and set the round timer (30 s / 60 s / 90 s).
  </Step>
  <Step title="Play">
    Pass the device to the acting player, start the timer, and let the chaos begin. Scores are tracked automatically across all rounds.
  </Step>
</Steps>
